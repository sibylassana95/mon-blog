WordPress : Tutoriel 1 ( installation )

18 juin 202220 juin 2022sibylassana95Wordpress

Installation en localhost

L’installation de WordPress est aussi simple que vous ne le croyez, pour ce faire il suffit de suivre les étapes suivantes  :

1 – Télécharger le seveur Wamp

Choisissez la version qui correspond à votre système d’exploitation : Windows 64 bits, Windows 32 bits, Linux…

Démarrez votre serveur Wamp depuis le menu démarrer, et assurez vous qu’il fonctionne correctement ( vous devez obtenir une icône  verte dans la barre des tâches )

 2 – Créer une base de donnée

 Le système wordpress fonctionne en php & mysql, pour cela son installation nécessite une base de donnée mysql !

Pour créer cette dernière il suffit de se rendre sur phpmyadmin du serveur wamp à l’adresse :

http://localhost/phpmyadmin/

Cliquez ensuite sur le lien en haut à gauche nommé Bases de données comme le montre la figure ci-dessous et choisissez comme interclassement des caractères utf8_general_ci ( ce jeux de caractère est compatible avec toutes les langues ! )

Donnez ensuite un nom à votre base de donnée par exemple mabase

3 – Télécharger l’archive du système WordPress du site : http://fr.wordpress.org/

Décompressez l’archive et mettez son contenu dans un dossier, nommez  le par exemple wordpress, mettez ensuite ce dossier à la racine du serveur ( dans le cas de Wamp sous Windows le chemin est   C:\wamp\www\wordpress\ )

 4 – Lancer l’installation du système wordpress :

Maintenant que vous avez tout préparé, vous pouvez lancer l’installation du système wordpress depuis votre navigateur en tapant l’adresse url : http://localhost/wordpress/ 

Vous obtenez la fenêtre suivante :

Cliquez alors sur le bouton Créer un fichier de configuration

Vous obtenez à ce moment la fenêtre suivante :

Vous voyez sans doute un bouton en bas intitulé : C’est parti !  Cliquez sur ce dernier  pour obtenir la fenêtre suivante :

Remplissez ensuite ce formulaire attentivement :

Nom de la base de donnée         :    mettez ici le nom de votre base de donnée qui est dans notre cas : mabase

Identifiant                                   :    vous mettez root ( en localhost c’est toujours root )

Mot de passe                              :   Vous laissez vide ( en localhost c’est toujours vide ! )

Adresse de la base de donnée   :  mettez localhost ( en serveur local c’est toujours localhost )

Préfix des tables                        :    Vous pouvez mettre une valeur ou bien laisser wp_ par défaut

Cliquez ensuite sur le bouton Envoyer  qui se trouve en bas de la fenêtre, s i tous les champs sont remplis  correctement vous obtenez la fenêtre suivante :

A ce niveau, une fenêtre s’affiche et vous demande de remplir quelque informations qui sont utiles pour continuez l’installation du système wordpress :

Remplissez soigneusement ce forulaire :

Cliquez ensuite sur le bouton en bas intitulé Installer WordPress pour obtenir la fenêtre suivante qui vous indique la fin d’installation du système WordPress  :

Vous pouvez maintenant vous connecter à la zone admin de votre site WordPress en cliquant sur le bouton Se connecter  pour obtenir une vue comme celle ci :

C’est votre zone admin WordPress, elle vous permet de gérer la totalité de votre site WordPress :  Articles, Médias, Pages, Commentaires …

Pour accéder à votre site WordPress depuis votre zone admin, pointez sur le nom de votre site à l’aide  de votre sourie sur le coin supérieur gauche, vous allez constater l’affichage  d’un menu intitulé   : Aller sur le site :

Cliquez sur ce menu pour accéder à votre site WordPress :

Toutes mes félicitations !

5 – Installation de WordPress en vidéo

Dans ce premier tutoriel de WordPress vous allez apprendre comment installer le système WordPress ainsi que toutes les étapes d’installation du CMS WordPress : téléchargement, décompression de l’archive wordpress, création d’une base de donnée en localhost sur Wamp et finalement l’installation du système wordpress.
