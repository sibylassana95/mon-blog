Les caractères spéciaux

18 juin 202218 juin 2022sibylassana95HTML

En HTML certains caractères ne sont pas visualisés correctement comme par exemple : ô, ï …

Afin de pallier au problème, ce dernier est doté d’un code spécifique pour chaque caractère spécial, par exemple :

&eacute;  designe  le code du caractère :    é

&eagrave;  designe le code du caractère  :   è

Ainsi pour écrire le code HTML de la phrase :  L’âge de l’élève est 17 ans

On utilise le code HTML  :

L’&acirc;ge de l’&aacute;l&agrave;ve est 17 ans
